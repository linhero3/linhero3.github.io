JSON Web Tokens

一. JWT 简介

JSON Web Tokens are an open, industry standard RFC 7519 method for representing claims securely between two parties.

JWT是一个标准，用来两方之间安全传输数据。JWT传输的数据其实就是Json web token字符串。所以广义上，JWT是一个标准的名称；狭义上，JWT指的就是用来传递的那个token字符串。这个串有两个特点：

紧凑：JWT字符串很小，能通过http请求参数、请求体以及http header的方式来传递；
自包含：JWT字符串可包含很多信息，如用户的ID、角色等，拿到该串，即可得到这些信息。

一. 浏览器同源策略

same origin

Two URLs are said to have the “same origin” if they have the same protocol, domain and port.The “Same Origin” (same site) policy limits access of windows and frame to each other.

如果两个window的一级域名相同，如 site.com, app.site.com, weapp.site.com，可以通过设置所有窗口的document.domain，来认为这些域名是同源的。

document.domain = 'site.com';

一. Maven简介

用maven来管理项目目前已是java web领域的共识。maven，一个开源项目管理工具，采用pom(Project Object Model)的概念，将所有配置信息定义在pom.xml中。
简单来说，maven有如下优点：

  • 简化项目依赖管理，Jar通过坐标即可引入，易于上手。
  • 统一的目录结构，统一的软件构建阶段。
  • 仓库保存构建，便于协同开发。
  • 多插件支持，拓展容易。
