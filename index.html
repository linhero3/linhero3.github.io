1. Linux 安装

Linux vs Windows

  • Linux严格区分大小写
  • Linux中所有的内容都是文件，包括硬件
  • Linux不靠扩展名区分文件类型
    以下扩展名约定俗成，便于阅读

    压缩包: .gz, .bz2, .tar.bz2, .tgz
    二进制软件包: .rpm
    网页文件:     .html
    脚本文件: .sh
    配置文件:     .conf

linux硬盘分区

  • 分区
    主分区最多4个,可包含一个扩展分区；扩展分区只能包含逻辑分区。
    必须分区：根分区，swap分区
  • 格式化：给分区写入文件系统(EXT4/NTFS…)
  • 挂载：给每个分区分配挂载点(win的盘符)（/ , swap, /boot）

    根分区必须： /
    swap分区必须： swap(交换分区/虚拟内存，内存2倍)
    引导分区推荐： /boot(启动分区）

  • 硬件设备文件名
    IDE硬盘: /dev/hd[a-p]
    SCSI/SATA/USB硬盘: /dev/sd[a-p]

    /dev/sda1: 第一块SATA硬盘的第一个主分区;
    /dev/sdb5: 第二块SATA硬盘的第一个逻辑分区
    用a/b/c字母标识硬盘，用数字标识分区(主分区:1-4 ; 逻辑分区:5开始 )

2. Linux常用命令

  • 命令基本格式
    命令 [选项] [参数]
    命令提示符：[root@arya ~]#

    root: 用户
    arya: 主机名
    ~: 当前目录(家目录 /root or /home/user_name)
    /#: 超级用户提示;普通用户$

    文件信息命令：ls -l

    -a 显示所有文件，包括隐藏文件
    -l 显示详细信息
    -d 查看目录属性
    -h 人性化显示文件大小(不以字节为单位)

    [root@arya ~]# ls -l
drwxrw-r-x root root 33333  4月 29 16:50  文件名

    drwxrw-r-x
    第一位: 文件类型（-文件 d目录 |软连接文件）
    2-10位: 文件权限(r读w写x执行 u所有者 g所属组 o其他人)

    root root 33333
    文件所有者 文件所属组 文件大小(字节)
