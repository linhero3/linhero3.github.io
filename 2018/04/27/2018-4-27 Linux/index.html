1. Linux 安装

Linux vs Windows

  • Linux严格区分大小写
  • Linux中所有的内容都是文件，包括硬件
  • Linux不靠扩展名区分文件类型
    以下扩展名约定俗成，便于阅读

    压缩包: .gz, .bz2, .tar.bz2, .tgz
    二进制软件包: .rpm
    网页文件:     .html
    脚本文件: .sh
    配置文件:     .conf

linux硬盘分区

  • 分区
    主分区最多4个,可包含一个扩展分区；扩展分区只能包含逻辑分区。
    必须分区：根分区，swap分区
  • 格式化：给分区写入文件系统(EXT4/NTFS…)
  • 挂载：给每个分区分配挂载点(win的盘符)（/ , swap, /boot）

    根分区必须： /
    swap分区必须： swap(交换分区/虚拟内存，内存2倍)
    引导分区推荐： /boot(启动分区）

  • 硬件设备文件名
    IDE硬盘: /dev/hd[a-p]
    SCSI/SATA/USB硬盘: /dev/sd[a-p]

    /dev/sda1: 第一块SATA硬盘的第一个主分区;
    /dev/sdb5: 第二块SATA硬盘的第一个逻辑分区
    用a/b/c字母标识硬盘，用数字标识分区(主分区:1-4 ; 逻辑分区:5开始 )

2. Linux常用命令

2.1 命令基本格式

命令 [选项] [参数]
命令提示符：[root@arya ~]#

root: 用户
arya: 主机名
~: 当前目录(家目录 /root or /home/user_name)
#: 超级用户提示;普通用户$

文件信息命令：ls -l

-a 显示所有文件，包括隐藏文件
-l 显示详细信息
-d 查看目录属性
-h 人性化显示文件大小(不以字节为单位)
-i 文件inode(每个文件的标识ID)

[root@arya ~]# ls -l
drwxrw-r-x root root 33333  4月 29 16:50  文件名

drwxrw-r-x

第1位: 文件类型（-文件 d目录 |软连接文件）
2-10位: 文件权限(r读w写x执行; 三组分别是: u所有者 g所属组 o其他人)
root root 33333: 文件所有者 文件所属组 文件大小(字节)

2.2 文件处理命令

  1. 目录/文件
    创建目录 : mkdir -p [目录名] make directories
    -p 递归创建 mkdir -p father/son

    切换目录 : cd change directories
    tab 目录补齐，提示

    删除目录 : rm -rf 文件或目录 remove
    -r 删除目录
    -f 强制(force)

    复制 : cp copy
    -r 复制目录
    -p 连带文件属性复制(时间、u/g/o … )
    -d 若文件是链接，复制链接属性
    -a 相当于-pdr

    剪切/改名 : mv copy
    跟cp不同的是，mv源文件不再存在

  2. 链接
    链接 : ln -s [源文件] [目标文件] link
    -s 创建软链接(类似快捷方式,与硬链接区别可查资料)

2.3 文件搜索命令
  1. 搜索文件: locate [文件名]
    在locate db(/var/lib/mlocate)数据库中搜索，不用全盘扫描，速度块。
    命令updatedb : 强制更新locate数据库(默认一天一更新)
    locate配置文件 : /etc/updatedb.conf

  2. 搜索命令: whereis [命令名] which [命令名]

    [root@arya mlocate]# whereis ls
ls: /usr/bin/ls /usr/share/man/man1/ls.1.gz /usr/share/man/man1p/ls.1p.gz
[root@arya mlocate]# which ls
alias ls='ls --color=auto'
/usr/bin/ls

    whoami whatis …

    PATH环境变量: 系统命令路径(类似windows的path)

    [root@arya mlocate]# echo $PATH
/usr/local/bin:/usr/local/sbin:/usr/bin:/usr/sbin:/bin:/sbin:/home/mrd/.local/bin:/home/mrd/bin
[root@arya mlocate]#

  3. find
    find [搜索范围] [搜索条件]

    3.1 -name 按名称搜索
    3.2 -iname 不区分大小写
    3.3 -user 按所有者搜索
    3.4 -mtime(atime/ctime) 按时间搜索
    3.5 -size 按大小搜索
    3.6 -a/-c 多条件搜索
    4.7 -exec [命令] {} \ 对搜索结果后处理

    按名称搜索 -name

    [root@arya /]# find / -name mrd
/usr/bin/mrd
/var/db/sudo/lectured/mrd
/home/mrd

    避免大范围搜索(耗资源)， 一般使用通配符匹配

    * 匹配任意内容
    ? 匹配任意一个字符
    [] 匹配’[]’内的任意一个字符

    按时间搜索 -[acm]time

    find /var/log/ -mtime +10   #查找10天前修改的文件
[root@arya /]# find /var/log/  -mtime +10
/var/log/pluto/peer
/var/log/speech-dispatcher

    -10 10天内；10 10天当天；+10 10天前
    atime 文件访问时间(access)； ctime 文件属性改变时间(change)； mtime 文件内容修改时间(modify)

    按大小搜索 -size

    [root@arya etc]# find /etc -size +2M
/etc/udev/hwdb.bin
/etc/selinux/targeted/policy/policy.30
/etc/selinux/targeted/active/policy.kern
[root@arya etc]# find /etc -size +2048k
/etc/udev/hwdb.bin
/etc/selinux/targeted/policy/policy.30
/etc/selinux/targeted/active/policy.kern

    +25k: 大于25k
    必须加单位: k(小写)/M(大写)

    find逻辑查找 -a/-o(与/或) 

    [root@arya etc]# find /etc -size +1k -a -size -3k

    find结果进行处理
    find语句 -exec [命令] {} \;

    [root@arya etc]# find /etc -size +5M -exec ls -lh {} \;
-r--r--r--. 1 root root 7.2M 4月  29 06:22 /etc/udev/hwdb.bin

  4. grep 搜索字符串
    grep [选项] 字符串 文件名

    -i 忽略大小写
    -v 排除指定字符串

    find vs grep

    find搜索文件；grep搜索文件内容
    find搜索文件名用通配符(*/?/[])匹配，并且是完全匹配；grep搜索文件内容用正则表达式匹配，并且是包含匹配。

2.4 帮助命令

man [命令]

man -f passwd -f命令等级
man 5 passwd

[命令] --help 选项帮助

2.5 压缩解压缩命令

1）压缩文件格式

.zip

zip [压缩文件名] [源文件]
zip -r [压缩文件名] [源目录]
unzip

.gz

gzip [源文件] 不保留源文件
gzip -c [源文件] > [压缩文件] 保留源文件
gzip -r [目录] 压缩目录下的文件，不能压缩目录
gunzip

.bz2

bzip2 [源文件]
bzip2 -k [源文件] 保留源文件
bzip2不能压缩目录
bunzip2 [压缩文件]

2）常用压缩解压缩命令
先打包文件，再压缩成相应的压缩文件格式

  • tar
    tar -cvf [打包文件名] [源文件]
    tar -xvf [打包文件名]
    options :

    -v 显示过程
    -f 指定打包后的文件名
    -c 打包
    -x 解包

  • .tar.gz
    tar -zcvf [压缩包名.tar.gz] [源文件]
    tar -zxvf [压缩包名.tar.gz]
    options :

    -z 压缩/解压缩tar.gz格式

  • .tar.bz2
    tar -jcvf [压缩包名.tar.bz2] [源文件]
    tar -jxvf [压缩包名.tar.bz2]
    options :

    -j 压缩/解压缩tar.bz2格式

2.6 关机/重启命令

shutdown
options

-c 取消前一个关机命令
-h 关机
-r 重启
